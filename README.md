## c++ basics

### data types
```c++
// numbers
int i = 0;
short s = 0;
long l = 0;
long long ll = 0;
float f = 0.0;
double d = 0.0;
long double ld = 0.0;
uint8_t u8 = 0;
uint16_t u16 = 0;
uint32_t u32 = 0;
uint64_t u64 = 0;
int8_t i8 = 0;
int16_t i16 = 0;
int32_t i32 = 0;
int64_t i64 = 0;


// bool
bool b = true;

// character
char c = 'a';
unsigned char uc = 255;
std::string s = "lightblue";

// array
int a[5] = {0, 1, 2, 3, 4};
int b[] = {3, 4, 5};
char s[] = "hello world";
int[][] = {{1,2}, {3,4}}

// pointer
char *p = "hello";
void *np = nullptr;

// reference
int x = 100;
int &r = x;

// enums
enum weekend {SAT, SUN};
```
